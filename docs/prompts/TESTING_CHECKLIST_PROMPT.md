# Testing Checklist Prompt

Use this after each module.

```text
Please create a testing checklist for the module you just built.

Include:
1. Functional test cases
2. Form validation test cases
3. Permission test cases
4. Mobile responsive test cases
5. PWA/mobile app-like UX test cases
6. Translation test cases Khmer/English
7. Theme test cases light/dark
8. Empty state test cases
9. Error state test cases
10. Edge cases
11. Expected result for each test case

Make the checklist simple and practical for manual testing.
```
