# Step 1 — Technology Stack Prompt

```text
Technology Stack:

Frontend:
- Next.js App Router
- Shadcn UI
- Tailwind CSS
- Redux Toolkit / RTK Query
- React Hook Form
- Zod
- Lucide React
- Khmer/English translation
- Light/dark theme
- PWA support

Backend:
- NestJS
- Prisma
- PostgreSQL
- REST API
- JWT authentication
- Role-based permission
- Controller / Service / Repository pattern

Translation:
- Support Khmer and English
- Use translation keys
- Do not hardcode UI labels, buttons, menu names, table headers, validation messages, empty states, or errors

Theme:
- Support light mode and dark mode
- Use Shadcn UI and Tailwind theme tokens
- Keep design consistent on desktop and mobile

PWA:
- The system should work nicely when installed as a mobile PWA
- Support app-style mobile layout
- Support safe-area padding
- Avoid desktop-only layout on mobile
```
