# Father Business Management System - AI Prompt Pack

This package contains step-by-step prompts and project documentation for building the Father Business Management System module by module with Code AI / Claude Code / Cursor / ChatGPT.

Recommended usage:
1. Open a new AI chat for each feature.
2. Paste the master context files first.
3. Paste the specific module prompt.
4. Ask the AI to review your existing structure before coding.
5. Save the feature completion summary after each module.
